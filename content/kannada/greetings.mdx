# Kannada Greetings :icon[Hand]

Greetings are a practical way to start speaking early.

## Common Phrases

| Kannada | Transliteration | Meaning |
| --- | --- | --- |
| ನಮಸ್ಕಾರ | namaskara | hello |
| ಹೇಗಿದ್ದೀರಾ? | hegiddira? | how are you? |
| ಧನ್ಯವಾದಗಳು | dhanyavadagalu | thank you |

:::tip
Use `ನಮಸ್ಕಾರ` for a respectful greeting in many everyday situations.
:::

## Practice

Reveal the meaning: ಧನ್ಯವಾದಗಳು means [[thank you]].
