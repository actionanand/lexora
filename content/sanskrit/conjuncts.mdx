Conjunct consonants appear when two or more consonants join closely. Read them slowly at first, then let the sound become one smooth unit.

:::tip[Reading Rhythm]
The halant removes the natural **a** sound. When consonants combine, listen for the joined sound rather than reading each letter separately.
:::

## Basic Patterns

| Joining letters | Conjunct | IAST | Sound |
| --- | --- | --- | --- |
| क् + ष | क्ष | kṣa | like **ksha** |
| त् + र | त्र | tra | like **tra** |
| ज् + ञ | ज्ञ | jña | like **gya / jnya** |
| द् + य | द्य | dya | like **dya** |
| क् + त | क्त | kta | like **kta** |
| त् + त | त्त | tta | like **tta** |

## Quick Check

1. Read each conjunct once by itself.
2. Read the same sound inside a word.
3. Reveal transliteration only if your pronunciation feels unsure.
4. Reveal the meaning after you can read the word smoothly.
